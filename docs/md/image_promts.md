# Global prompt rules for all images
- style: realistic editorial photography
- mood: quiet luxury, caring expert, calm feminine confidence
- palette: ivory, warm beige, greige, taupe, mocha, soft caramel accents
- lighting: soft natural daylight
- avoid: clinical interiors, aggressive beauty retouching, flashy props, oversaturated colors, glam makeup, plastic skin, stock-photo smiles
- composition: vertical, generous negative space for website layout
- skin: natural texture, believable beauty, healthy not overprocessed
